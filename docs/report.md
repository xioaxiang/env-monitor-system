# 环境监测数据分析系统 - 项目报告

## 一、项目概述
### 1.1 项目背景
本项目旨在通过Python+Git实现环境传感器数据的自动化分析，为环境质量评估提供数据支撑，同时掌握Git团队协作流程。

### 1.2 项目目标
- 掌握Git版本控制和GitHub协作流程；
- 实现环境数据的加载、清洗、分析与可视化；
- 完成团队分工与代码集成。

### 1.3 技术栈
- 编程语言：Python 3.9
- 数据处理：Pandas、NumPy
- 数据可视化：Matplotlib、Seaborn
- 版本控制：Git + GitHub

## 三、实现过程

### 3.1 数据说明
- 数据来源：模拟传感器数据
- 时间范围：2024年1月1日 - 1月30日
- 监测指标：温度、湿度、PM2.5
- 监测站点：站点A、站点B

### 3.2 分析结果

#### 3.2.1 基础统计
| 指标 | 均值 | 中位数 | 标准差 | 最小值 | 最大值 |
|------|------|--------|--------|--------|--------|
| 温度(℃) | 25.2 | 25.1 | 4.8 | 14.2 | 36.5 |
| 湿度(%) | 60.3 | 59.8 | 9.5 | 38.5 | 84.2 |
| PM2.5 | 48.6 | 47.2 | 18.3 | 12.5 | 98.4 |

#### 3.2.2 空气质量评估
- 优：8天
- 良：18天
- 轻度污染：4天
- 优良天数比例：86.7%
